For better resilience to troubles, the deployment of the Galaxy server dedicated to our users is now automated using the GalaxyKickStart Ansible Playbook.

Therefore, the users with an account on this server may request any tool installation by raising a pull request after editing the artbio_tool_list.yml on our GitHub GalaxyKickStart repository.
