Mississippi[2] major upgrade. Sat 2021/09/11, 2021, 12:05.

We have experienced serious issues during a major server upgrade. As a result Mississippi[2] was barely available during September 5-10, 2021. We deeply appologize for the inconvenience caused to our Users.

Mississippi[2] is BACK and running NORMALY and, if the computer gods are more lenient, we do not expect maintenance operations impacting our users for a long time.

Please, note that tool installation in Mississippi[2] is now fully automated using Ansible. Therefore Mississippi users may request tool installation by raising a pull request after editing the mississippi_tool_list.yml on our GitHub GalaxyKickStart repository.
